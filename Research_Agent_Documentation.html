Research Agent

AI-Powered Academic Paper Tool
Technical Documentation - How It Works

Overview

Research Agent is a desktop application that helps researchers work with academic papers. It uses AI (Claude or ChatGPT) to automate time-consuming research tasks while keeping the human in control.

Four Main Features:

  • Paper Finder - Search for papers online using AI web search
  • PDF Analyzer - Ask questions about your local PDF collection
  • Citation Verifier - Check if citations support your claims
  • Data Extractor - Extract and verify specific data for meta-analysis

1. Paper Finder

What it does:

Searches the internet for academic papers matching your query. AI analyzes your search terms, finds relevant papers from academic sources, filters out loosely related results, and returns detailed information including titles, authors, summaries, and links.

4. Data Extractor (with Human Verification)

What it does:

Extracts specific data from multiple papers for meta-analysis. You define what data you need (e.g., "oxygen penetration depths"), AI scans all papers and extracts values with their context. Critically, every extracted data point includes the exact quote and page number, and YOU verify each item before it goes into the final table. This ensures 100% human-verified data for your meta-analysis.

Key Capabilities:

  • Extract numerical values, ranges, and conditions from text
  • Read and extract data from tables within PDFs
  • Extract data from figures (if text labels present)
  • Capture units, methods, sample types, and experimental conditions
  • Provide exact quotes and page numbers for verification
  • Human approval required for each data point
